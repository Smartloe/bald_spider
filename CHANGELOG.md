# 更新日志

本文档记录了 Bald Spider 的重要变更。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [0.1.0] - 2024-11-24

### 🎉 首次发布

#### ✨ 新增
- 基础异步架构设计
- 核心组件：Engine、Scheduler、Downloader、TaskManager
- 请求和响应数据结构
- 配置管理系统
- 爬虫基类和生命周期管理
- 并发控制机制
- 优先级队列支持
- 完整的项目文档

#### 🔧 架构特性
- 基于 asyncio 的异步编程模型
- 模块化设计，组件职责清晰
- 生成器模式支持内存友好的数据处理
- 信号量控制并发请求数量
- 优先级队列支持重要请求优先处理

#### 🛠️ 开发工具
- 使用 uv 进行现代 Python 包管理
- pytest 测试框架
- black 代码格式化
- 完整的 pyproject.toml 配置

#### 📚 文档
- 完善的 README.md 包含快速开始指南
- 架构文档详细说明系统设计
- 开发指南提供贡献规范
- 技术栈文档记录设计决策
- 项目概述展示发展路线图

#### 🧪 示例
- 百度爬虫示例展示框架使用方法
- 多爬虫项目示例
- 异步并发处理演示

### 📝 备注
这是一个早期版本，主要展示了框架的核心架构和基础功能。建议开发者在生产环境中使用时进行充分测试。

## 版本命名规范

我们遵循语义化版本控制 (Semantic Versioning)：

- **主版本号**：当你做了不兼容的 API 修改
- **次版本号**：当你做了向下兼容的功能性新增
- **修订号**：当你做了向下兼容的问题修正
